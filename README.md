# 🌸 Iris Flower Classification Web App

An interactive Machine Learning web application that predicts the species of an Iris flower based on its sepal and petal measurements.

---

## 🚀 Live Demo

👉 [Click here to view live app](https://iris-flower-classify.streamlit.app/)

---

## 📌 Features

- Predicts Iris flower species:
  - Iris Setosa
  - Iris Versicolor
  - Iris Virginica
- User-friendly web interface built with Streamlit
- Instant predictions with confidence scores
- Interactive input sliders
- Clean and responsive design

---

## 🛠️ Technologies Used

- Python  
- Streamlit  
- Scikit-learn  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Plotly  
- Joblib  

---

## 📊 Dataset

This project uses the famous **Iris Dataset**, which contains:

- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width
